# ADGM RAG Project

A Retrieval-Augmented Generation (RAG) system for querying the ADGM rulebook with real citations.

## Features
- PDF/DOCX ingestion
- Text chunking
- Embedding with FAISS vector store
- Query answering with LLaMA or other LLMs
